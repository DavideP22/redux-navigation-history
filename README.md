# react-redux-history

### Setup

Download [Node.js LTS (20.11.0)](https://nodejs.org/en)

Install pnpm:
```shell
$ npm i -g pnpm
```
Clone project:
```shell
$ git clone https://github.com/DavideP22/redux-navigation-history.git
```
Install dependencies:
```shell
$ cd react-redux-history-demo
$ pnpm i
```
Run the app:
```shell
$ pnpm dev
```
